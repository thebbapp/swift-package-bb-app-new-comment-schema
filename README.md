# ``BbAppNewCommentSchema``

Data structures for new comment creation.

## Overview

The BbAppNewCommentSchema library defines the data structures and schemas used in new comment creation workflows. It provides type definitions for comment creation requests, validation rules, reply threading data, and the data models that coordinate between the new comment UI and the underlying content sources.
